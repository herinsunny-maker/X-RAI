# X-RAI
This project is an AI-powered system that detects and classifies the severity of knee osteoarthritis from X-ray images using the EfficientNet deep learning model. It incorporates Grad-CAM visualization to highlight the regions of the X-ray that contribute most to the prediction, providing an interpretable and reliable diagnostic aid for healthcare professionals.
## Features
- Detects knee osteoarthritis from X-ray images.
- Classifies osteoarthritis severity grades.
- Uses EfficientNet for image classification.
- Generates Grad-CAM visualizations.
- Displays prediction confidence scores.
# How to Use
- Launch the application.
- Upload a knee X-ray image.
- The model processes the image and predicts the osteoarthritis grade.
- View the confidence score and Grad-CAM heatmap.
- Analyze the highlighted regions to understand the model's decision.
